# Three-minute demo script

This recording uses the synthetic Maya Laurent cultural-production campaign.
It is one continuous non-IT trace: discovery, ranking, five tailored application
packages, PDF supervision, Chrome handoffs and sandbox receipts. No employer
application is submitted.

## Before recording

- Start Studio with the checked synthetic state and keep the campaign page open.
- Present the video as a verified replay of a completed campaign. Keep the real
  latency and token telemetry visible; do not imply that generation is instant.
- Install `jobauto@jobauto-studio` and verify that the Codex Chrome Extension can
  access local file URLs.
- Keep one completed run, its original CV, tailored CV, letter and one handoff
  ready in separate tabs.
- Record at 1080p with readable browser zoom and microphone audio.
- Use English narration and keep the final video strictly under three minutes.
- Do not show local paths, terminal secrets, personal browser data, unrelated
  tabs, real employer names or third-party logos. Use no copyrighted music.

## What the recording must make obvious

1. The input is a candidate-owned CV and policy, not a built-in profile.
2. Search is a batch operation with visible selection and rejection decisions.
3. The strategy is planned before writing; the rendered documents are inspected
   separately by the final supervisor.
4. The original CV belongs to the immutable run snapshot.
5. Chrome receives the exact hash-approved files through the installed Codex
   plugin.
6. `sandbox_verified` proves browser control, uploads and receipt persistence,
   not a third-party employer submission.
